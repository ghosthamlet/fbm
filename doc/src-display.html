SRC-DISPLAY:  Print the parameters of a source location model.

Src-display prints the parameters of the sources, and of the detector
and flow models (if any).

Usage:

    src-display log-file [ index ]

The parameters stored under the specified index are displayed.  If no
index is specified, the parameters in the last iteration stored in the
log file are displayed.
